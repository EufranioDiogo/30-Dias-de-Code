# 30 Dias de Code

Este é um repo que eu suponho fazer 30 days de code(programando), desde o básico **Hello World** até a coisas mais complicadas.

## Estrutura

Para todos os dias será criado uma pasta onde vai conter a descrição do exercícios e os código para a solução dos problemas.

## De onde eu pego os exercícios?

Pego os exercícios apartir de uma plataforma online chamada <a href="https://hackerrank.com" target="_blank">Hackerrank</a>.

## Qual é o meu objectivo com estes 30 dias de code?

Voltar a estudar ou consolidar os conhecimentos técnicos de algoritmos e estruturas de dados para manter eles afiados.


## Linguas do repo

Em primeiro lugar colocarei os READMEs dos exercícios em **EN** e também **PT**.
